The Implementation of motorcycle security system using voice command and fingerprint sensors based on IoT


This project's made for motorcycle security system that possible to user feels savety of their motorbikes. Using Arduino Mega for the controller system and Blynk as an IoT framwork are make it easy for user to control their motorcycle. Through this technology, motorcycle owners do not have to worry about cases of motorcycle theft. because they can monitor the state of their motorcycle via smartphone remotely

other than that, motorcycle owners don't need to worry because the process of turning on a motorcycle is through fingerprints that cannot be imitated by others. This project is useful to use and it can be implemented for daily life.
